Image Component

An image is a graphic representation of something. Images can be made responsive (so that they don't jump out of its parent container), have rounded corners or can look like a circle.

Responsive Images

Images are made responsive with the class ".img-responsive".
max-width: 100%; and height: auto; are applied to the image so that it scales with the parent element.



Rounded Corners Images

The ".img-rounded-corners" class adds rounded corners to an image.



Circle Images

The ".img-circle" class shapes the image to cicle.
