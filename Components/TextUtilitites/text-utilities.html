Font Family

SNKRS UI CSS Library uses Montserrat, as its default font, which is a free font available at Google Fonts.


Alignments

Use classes ".text-center" , ".text-left" or ".text-right" to center, left or right align text.

Font Weight

Use classes ".fw-300", ".fw-400", ".fw-500" , ".fw-600", ".fw-700" or ".fw-900" to change the font-weight of text ranging from 300 to 900.

Other Text Styles

  • To add strike-through on text, use ".text-strike" class
  • To add a line under a text, use ".text-underline" class
  • To add different text colors, use one of the classes ".text-error-color", ".text-warning-color" or ".text-success-color".
